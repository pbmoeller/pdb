# PDB - Philipp's Debugger

This repository contains PDB a simple debugger for C++ programs and is based on the internal library `libpdb`.

The core code in this repository is based on the book "Building a Debugger" by Sy Brand. It will be extended and modified over time.

# Dependencies

To build and use PDB, you will need the following library dependencies:
- libedit
- catch2
